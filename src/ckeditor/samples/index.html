Developer's Guide

The most important resource for all developers working with CKEditor, integrating it with their websites and applications, and customizing to their needs. You can start from these:

  • Adding CKEditor - How to integrate the editor with your website or application using various available methods.
  • Advanced Content Filter - How CKEditor filters input data and adjusts your content to your configuration.
  • Advanced Installation Concepts - How to upgrade, install additional components (plugins, skins), or create a custom build.
  • Editor Configuration - How to configure the editor and where to add custom configuration code.
  • Saving editor data - How to work with editor input and output.

When you have the basics sorted out, feel free to browse some more advanced sections like:

  • Functionality Overview with descriptions and samples for various editor features.
  • Plugin SDK, Widget SDK, and Skin SDK - Useful when you want to create your own editor components.
