Customize Your Editor

Modular build and numerous configuration options give you nearly endless possibilities to customize CKEditor. Try to add the following code to your config.js file and refresh this page!

config.language = 'es';
config.uiColor = '#F7B42C';
config.height = 300;
config.toolbarCanCollapse = true;
